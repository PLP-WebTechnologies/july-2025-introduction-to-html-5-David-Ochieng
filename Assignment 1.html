Page Structure Elements

<header>

Contains the page title and introduction. See the top of this page.

<nav>

Navigation links. The menu above uses this element.

<main>

Primary content area. You're reading inside the main element now.

<section>

Groups related content. Each topic here is a separate section.

<footer>

Page information and links. Check the bottom of this page.

Content Elements

<article>

Self-contained content. Each element description is an article.

<aside>

This aside contains supplementary information related to the main content.

Headings (H1-H6)

Create content hierarchy:

  • H1: Main page title
  • H2: Section headings
  • H3: Subsection headings

Accessibility Benefits

Screen Reader Support

Semantic elements create landmarks that help users navigate with assistive technology.

SEO Advantages

Search engines understand page structure better with proper semantic markup.
